Projects

Java-Based Fruit Store PoS System

As part of my 2-1 semester, my teammates and I developed a Fruit Store Point-of-Sale (PoS) System using Java. We built this project in Eclipse, utilizing JOptionPane from the Java Swing library to create an intuitive graphical interface. The system efficiently manages customer and owner transactions, allowing for features such as adding/removing fruits, managing inventory, applying discounts, and maintaining membership points for customers.

Restaurant Management System (SQL Project)

During my 2-2 semester, my teammate and I developed a Restaurant Management System using MySQL. This database-driven application streamlines restaurant operations, including menu management, order processing, payment handling, reservations, and staff coordination. The system enhances efficiency, minimizes billing errors, and helps prevent food shortages, making it a valuable asset for restaurant owners.
